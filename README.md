## What it does

Multi-phase research framework for commercial real estate. Runs market rate comparisons with source citations, tenant prospecting strategies by property type, and competitive intelligence gathering.

## Features

- Market rate comparisons with source citations
- Tenant prospecting strategies by property type
- Competitive intelligence gathering
- Supports Industrial, Flex, Office, Retail, and Multifamily

## Install

```
claude plugins install cre-property-research@not-my-job
```
